This package wraps SQLite, which is in the public domain. The SQLite
source code itself is not covered by this license — see https://sqlite.org
for SQLite's own terms (which are: there are none, do whatever you want).

The components covered by this MIT license are the wrapper layer between
VB6 and SQLite, including:

  - sqlite3_vb6.c       (C stdcall shim)
  - sqlite3_vb6.def     (export definitions)
  - modSQLite.bas       (VB6 declares and helpers)
  - cSQLite.cls         (connection class)
  - cSQLiteStatement.cls (prepared statement class)
  - cSQLiteResults.cls  (result set iterator)
  - cSQLiteSchema.cls   (schema introspection and DDL)
  - cSQLiteColumn.cls   (column metadata)
  - cSQLiteAPI.cls      (global low-level API surface)
  - cFileRecord.cls     (record helper used by the export form)

  and any other VB6 / Python source files distributed alongside this
  LICENSE file.
